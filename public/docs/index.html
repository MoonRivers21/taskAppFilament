Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

POST api/tasks

Example request:
curl --request POST \
    "http://localhost:8000/api/tasks" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"title\": \"ltdjc\",
    \"content\": \"fjlijrjmblqeztgyptrbjyjsrqspbjecqqumnuf\",
    \"user_id\": 4
}"

const url = new URL(
    "http://localhost:8000/api/tasks"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "ltdjc",
    "content": "fjlijrjmblqeztgyptrbjyjsrqspbjecqqumnuf",
    "user_id": 4
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

URL Parameters

id   string   

The ID of the task. Example: tempore

Body Parameters

Must be at least 3 characters. Example: ajpnwpuyzhafxpkdvupaqdkyzycaurvpqxooiucpuhhcizzrmvxaozvfcfzpakcoeuibarhmxirebssyhzrpoij

image   string  optional  
published   string  optional  
publishedAt   string  optional  
status   string  optional  

Example: To-do

Must be one of:
  • To-do
  • In-progress
  • Done
